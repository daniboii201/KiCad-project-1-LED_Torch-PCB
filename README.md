# KiCad project #1: LED Torch PCB
This is my first PCB design created using KiCad. It’s a beginner-friendly project featuring a very simple circuit: an LED, a resistor, a push button, and a CR2032 battery holder.

## Features
• LED torch powered by a CR2032 coin cell

• Ideal for learning PCB design basics: schematic capture, footprint assignment, and routing

### Components:
• LED

• Resistor

• Push Button

• CR2032 Battery Holder

### Characteristics:
• Copper on top layer

• Edge cuts with rounded segments

• Combination of surface-mounted and through-hole components

• Includes a mounting hole

• Silkscreen and graphics on both front and back
